# 🎬 Cinema Scraper - Maringá

Script em Python que realiza web scraping do portal de cinema de Maringá, extraindo filmes em cartaz e seus respectivos horários, armazenando os dados de forma estruturada em um banco de dados SQLite.

##  Tecnologias
- **Python 3**
- **BeautifulSoup4** (Extração de dados HTML)
- **Requests** (Protocolo HTTP)
- **SQLAlchemy** (ORM para persistência em banco de dados)

##  Como executar
1. Instale as dependências:
   ```bash
   pip install -r requirements.txt

##  Iniciando:
   python coleta_dados.py
